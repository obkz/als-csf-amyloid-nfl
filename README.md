# Analysis scripts for ALS CSF biomarker study

This repository contains R scripts used for the statistical analyses and visualization in our manuscript:

**"Complementary roles of cerebrospinal fluid amyloid and neurodegeneration biomarkers in the diagnosis and prognosis of amyotrophic lateral sclerosis"**

## Repository contents

* `scripts_pub/`: R scripts for statistical analyses and figure/table generation
* `README.md`: overview of the repository
* `.gitignore`: file exclusion rules for data and output files

## Notes

The scripts are shared to improve transparency regarding the analytical workflow. They are not intended to provide a fully executable reproduction package without access to the underlying study data.
Individual-level clinical and biomarker data are not included because of institutional and legal restrictions on patient data confidentiality.

The manuscript citation and DOI will be added upon publication.
